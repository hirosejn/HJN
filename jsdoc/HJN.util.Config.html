new Config(prefixopt, olopt)

Parameters:
  • prefix String <optional> ''
    定数の名前空間を一位に指定する文字列、指定しない場合グローバル
  • ol String <optional> 'ol'
    インデント(.nDown() .nUp())に使うHTMLタグ

         .label(null,"------").n() // ラベルを表示し、改行

         .radio(def("ENDIAN_BIG", false), null, "big");
