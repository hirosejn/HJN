Type Definitions

CONC

CONC:多重度の時系列データ
[{x:変化時刻(ms) ,y:多重度数 }]
Type:
  • array.<xMs, yInt>

CTPS

CTPS: 秒間最大多重度の時系列データ
[{x:秒毎時刻(ms), y:秒内最大多重度数 }]
Type:
  • array.<xMs, index>

EAPS

EAPS: 秒間平均TAT（応答時間）時系列データ（終了時刻ベース）
[{x:秒毎時刻(ms), y:秒内平均レスポンス(sec) }]
Type:
  • array.<xMs, ySec>

EMPS

EMPS: 秒間最大TAT（応答時間）時系列データ（終了時刻ベース）
[{x:秒毎時刻(ms), y:秒内最大レスポンス(sec) }]
Type:
  • array.<xMs, ySec>

ETAT

ETAT: 終了時刻のTAT（応答時間）時系列データ
[{x:変化時刻(ms) ,y:レスポンス(sec), fileIdx:ファイル配列位置, pos:レコード位置, len:レコード長, sTatIdx: sTatの配列位置 }]
Type:
  • array.<xMs, ySec, index, index, number, index>

ETPS

ETPS: 秒間終了件数の時系列データ
[{x:秒毎時刻(ms), y:秒内終了件数 }]
Type:
  • array.<xMs, yInt>

index

index: 配列の位置（インデックス）に用いる整数
Type:
  • number

seriesSet

seriesSet: dygraph用時系列データ配列
Type:
  • array.<CONC, CTPS, ETPS, STAT, ETAT, EMPS, EAPS>

STAT

STAT: 開始時刻のTAT（応答時間）時系列データ
[{x:開始時刻(ms) ,y:レスポンス(sec), eTatIdx: eTatの配列位置 }]
Type:
  • array.<xMs, ySec, index>

xMs

xMs: x軸に用いる時刻(ms)
Type:
  • number

yInt

yInt: y軸に用いる整数（個数など）
Type:
  • number

ySec

ySec: y軸に用いる秒単位の時間（応答時間など）
Type:
  • number
