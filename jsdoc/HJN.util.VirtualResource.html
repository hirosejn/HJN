new VirtualResource(nameopt, capacityopt, timeoutopt, queueDepthopt, queueWaitopt)

VirtualResource
Parameters:
  • name String <optional> "unlimited"
    リソース名（"unlimited"はリソース解放待ちを管理しない）
  • capacity Number <optional> 1.0
    保有リソース総量（数）
  • timeout Number <optional> 10秒
    処理のタイムアウト時間（未使用）
  • queueDepth Number <optional> Number.MAX_SAFE_INTEGER
    リソース取得待ちキューの深さ（数）、キュー溢れ時は即時エラー終了しリソース処理しない
  • queueWait Number <optional> 10秒
    最大キュー滞留時間（リソース取得待ちタイムアウト時間）
